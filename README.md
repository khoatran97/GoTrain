# GoTrain
Hệ thống quản lý bán vé xe lửa - Đồ án cuối kì Phân tích Thiết kế Phần mềm

### Cấu trúc thư mục
```
--- public          (chứa các file dùng chung)   
|   |--- css
|   |--- fonts
|   |--- icons
|   |--- images
|   |--- js
|   |--- sass
|
--- views           (chứa các view)
|    |--- layouts   (chứa layout chung)
|
--- service.js      (file dịch vụ)
```
